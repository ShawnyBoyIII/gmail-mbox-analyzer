
## 2024-05-24 - explicit copy to clipboard buttons
**Learning:** For actions where the primary purpose is generating text to be used elsewhere (like search queries), auto-selecting text is not enough. Users still have to discover and rely on keyboard shortcuts (`Ctrl+C` or `Cmd+C`) or context menus.
**Action:** Always provide explicit, one-click "Copy to Clipboard" buttons with visual confirmation ("Copied!") when generating copy-paste content to reduce interaction friction.
