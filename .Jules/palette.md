## 2024-05-15 - Explicit Copy-to-Clipboard Button
**Learning:** For popups generating copy-paste content (like Gmail search queries), relying solely on users selecting text and pressing Ctrl+C/Cmd+C introduces unnecessary friction, especially when the text block is large or the user is not a power user.
**Action:** Always provide an explicit, one-click "Copy to Clipboard" button with temporary visual confirmation (e.g., "Copied!") next to generated text blocks that are meant to be copied.
