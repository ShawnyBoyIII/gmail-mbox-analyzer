## 2026-06-07 - Add one-click copy to generated text
**Learning:** When generating text that a user is intended to copy (like a search query or filter string), forcing them to manually highlight and use keyboard shortcuts creates unnecessary friction, especially on touch interfaces or for less tech-savvy users.
**Action:** Always include a dedicated, prominent 'Copy to Clipboard' button with visual feedback (like changing text to 'Copied!') when presenting generated text intended for use elsewhere.
